## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/GoBookstore.git
   ```
2. Navigate to the project directory:
   ```bash
   cd GoBookstore
   ```
3. Install dependencies:
   ```bash
   go mod tidy
   ```
4. Set up the MySQL database

5. Run the application:
   ```bash
   go run main.go
   ```
